How Beerpay Works

Givers

Use beerpay to motivate others to continue developing, also you can set Wishes (e.g. new features) for any projects and then use it!

  • Support Projects
  • Support existing Wishes
  • Make Wishes for new features
  • Make Wishes to fix issues

Receivers

Add our badge in your projects on Github, even if was forgotten in time, and you will be able to get Support or Wishes (e.g. new features) from the OS community.

  • Import your projects from GitHub
  • Add your custom badges
  • Improve projects through wishes
  • Start to get support
